# Ais & Tangkas — Cinematic Wedding Invitation
**Next.js 16 · TypeScript · Mobile-First**

## Jalankan
```bash
npm install
npm run dev
```
Buka: http://localhost:3000

## Navigasi
- **Swipe atas/bawah** di HP — snap ke scene berikutnya
- **Scroll** di desktop — snap per section
- **Titik emas** di kanan layar — tap/klik langsung ke scene manapun
